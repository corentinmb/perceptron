This is a java project for extracting drugs, diseases and their CID relations via perceptron.
The program is tested on java runtime environment 7.
All the copyrights of data, libraries and tools belong to their authors.  

The architecture of this project is shown as below:
lib --- all the third-party libraries and tools
data --- all the data and dictionaries
src --- all the source code
		joint --- model for jointly extracting drugs, diseases and their CID relations
		perceptron --- model for extracting drugs, diseases
		utils --- relevant utility classes
